# AI占卜大师

## 项目恢复说明

从 APK 反编译恢复的 Capacitor 占卜应用。

### 功能模块
- **八字分析** - 日干、大运、流年
- **六爻占卜** - 卦象解读
- **奇门遁甲** - 排盘解读
- **紫微斗数** - 命盘分析
- **面相学** - 面部特征分析

### 知识库
包含丰富的占卜知识数据：
- bazi_kb.json - 八字基础
- gua_kb.json - 卦象
- qimen_kb.json - 奇门
- ziwei_kb.json - 紫微斗数
- 以及更多专业资料

### 技术栈
- 前端: HTML5 + CSS + JavaScript
- 移动框架: Capacitor

### 运行
```bash
npm install
npm run dev
```
